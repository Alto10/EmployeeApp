# EmployeeApp
Employee Management Web App dengan React &amp; Node.js
## Teknologi
- Frontend: React, Bootstrap
- Backend: Node.js, Express
- Database: SQL Server
- Tools: Git, GitHub, Node.js, NPM

## Struktur Folder
EmployeeApp/
├─ frontend-react/
├─ backend/
├─ database/ # EmployeeDB.sql
└─ README.md

## Cara Menjalankan

1. Clone repository
   git clone https://github.com/username/EmployeeApp.git
   cd EmployeeApp

2. Backend
   cd backend
   npm install
   node server.js
   
3. Frontend
   cd frontend-react
   npm install
   npm start

4. Database
   Jalankan database/EmployeeDB.sql di SSMS untuk membuat database, tabel, dan data sample

**Catatan**
Jangan upload node_modules/ dan file .env
Jalankan backend sebelum frontend agar API bisa diakses
